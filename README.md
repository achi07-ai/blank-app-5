📅 マネたいむ。

〜 時間とお金をスマートに管理する、あなた専用のライフコンシェルジュ 〜

🔗 「マネたいむ。」を使ってみる
https://moneytime.streamlit.app/

「マネたいむ。」は、学習・バイト・プライベートの予定を一つのカレンダーで管理し、バイトの時給・固定給設定から今月の見込み給与を自動算出する、Streamlitベースの高機能スケジュール管理アプリです。
✨ 主な機能
1. 🗓️ ハイブリッド・カレンダー

    直感的な操作: 月・週・日表示の切り替え、ドラッグ＆ドロップによる予定の移動・時間調整が可能。

    視認性重視のデザイン: カテゴリ別の色分け、完了済みタスクのグレーアウト機能を搭載。

    全時間帯対応: 深夜から早朝まで、0:00〜24:00のすべての時間帯をシームレスに表示。

    月間ブロック表示: 月表示（Month）でも週表示と同様に色付きのブロック形式で予定を表示。


2. 💰 インテリジェント給与計算

    ハイブリッド計算: バイトの「時給」と「月間固定給」の両方に対応。

    リアルタイム算出: カレンダーの「バイト」予定から今月の勤務時間を集計し、内訳（固定給+時給分）とともに見込み給与を表示。

    設定の永続化: 入力した時給や固定給はSupabaseに保存され、再ログイン後も維持されます。

3. 🔔 スマート・リマインダー

    自動期日計算: カテゴリ（テスト・課題・遊び・日用品等）に応じた通知日を自動算出。

    ダッシュボード通知: 期限が近い未完了タスクがある場合、ログイン後の画面上部にアラートを表示。

4. 🔐 ユーザー管理

    Supabase Auth: セキュアなログイン/新規登録、およびログイン後のパスワード変更機能を完備。

🚀 今後の改善案（ロードマップ）

本プロジェクトのさらなる拡張として、以下の機能を計画しています。

    👥 複数人での予定共有機能: 家族やグループ間でのカレンダー共有・共同編集機能。

    📊 給料・勤務時間の統計（グラフ表示）: 月別収入の推移やカテゴリ別の時間配分を可視化する分析ダッシュボード。

    📲 外部通知連携: LINEやSlackへのリマインダー通知送信機能。

    🌙 テーマカスタマイズ: ダークモード対応や、ユーザー好みのカラーテーマ設定。

🛠️ 技術スタック

    Frontend: Streamlit

    Database / Auth: Supabase (PostgreSQL)

    Calendar Library: streamlit-calendar (FullCalendar)

    Timezone: pytz (Asia/Tokyo 完全対応)

⚙️ セットアップ方法
1. データベース（Supabase）の準備

SQL Editorで以下のテーブルを作成してください。
SQL

-- 予定管理テーブル
CREATE TABLE todos (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  user_id UUID REFERENCES auth.users(id) NOT NULL,
  title TEXT NOT NULL,
  category TEXT NOT NULL,
  start_at TIMESTAMP WITH TIME ZONE NOT NULL,
  end_at TIMESTAMP WITH TIME ZONE,
  reminder_at DATE,
  is_complete BOOLEAN DEFAULT FALSE
);

-- 給与・設定保存テーブル
CREATE TABLE settings (
  user_id UUID REFERENCES auth.users(id) PRIMARY KEY,
  hourly_wage INT DEFAULT 1200,
  fixed_salary INT DEFAULT 0
);

2. 依存ライブラリの設定

requirements.txt に以下を記述してください。
Plaintext

streamlit
supabase
streamlit-calendar
pytz

3. 環境変数の設定

Streamlit CloudのSecretsにAPI情報を設定します。
Ini, TOML

url = "YOUR_SUPABASE_URL"
key = "YOUR_SUPABASE_ANON_KEY"

💡 開発におけるこだわり

    UXの最適化: ダイアログ内での編集・削除、カレンダー内での改行表示など、スマホ・PC両方での使い勝手を追求。
